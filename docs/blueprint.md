# **App Name**: Hiigsi skills

## Core Features:

- User Authentication: Secure email and password authentication with optional Clerk integration, assigning default roles upon signup and JWT/session validation for protected routes and APIs.
- Role-Based Access Control (RBAC): Implementation of RBAC to manage user permissions and access levels within the system.
- User Management: CRUD operations for managing users, roles, and permissions, including staff, parents, and students accounts, with activation/deactivation functionality and user activity/audit logs.
- Academic Management: Management of classes/grades, sections, subjects, teacher assignments, student enrollment, academic calendar, class timetable, syllabus management, attendance tracking, exams & grading, and report cards.
- Student Management: Student profiles with guardian/parent linking, admission management, promotion & graduation features, attendance tracking, and academic history.
- Fee Structure Creation with AI Assistance: Define various school fees based on grade, level, student, and historical financial data. An AI tool will assist the administrator in proposing tuition, transport, meals, and accommodation fees. AI uses location, cost of services and economic index data to generate informed proposals for new or revised school fees.
- Finance & Fees: Management of fee structures, student invoicing, payment tracking, discounts and scholarships, expense management, and financial reports.
- Communication System: Platform for announcements, in-app and email notifications, and parent-teacher communication.
- Dashboard Analytics & Reports: Display school KPIs, create and manage school report cards, system reports, and exports in PDF, Excel, and CSV formats.

## Style Guidelines:

- Primary color: A serene blue (#64B5F6) to evoke trust and knowledge.
- Background color: A light blue (#E3F2FD), gently desaturated, providing a calm backdrop that keeps the focus on content.
- Accent color: A muted purple (#9575CD), adds a touch of creativity, contrasting yet harmonious with the primary.
- Body and headline font: 'PT Sans' (sans-serif) for a modern, clean and slightly warm feel suitable for both headlines and body text.
- Consistent and clear icons from a set like Material Icons to represent actions and categories.
- A clean, grid-based layout that is responsive and adapts to different screen sizes, ensuring usability on all devices.
- Subtle transitions and animations to provide feedback on user interactions and guide the user through the interface.
